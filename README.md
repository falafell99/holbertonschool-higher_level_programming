# Higher Level Programming

## Description
This repository contains projects and exercises for the Higher Level Programming track at Holberton School. The curriculum covers various programming languages and concepts including Python, SQL, and more.

## Directory Structure

### SQL_introduction
- **0-list_databases.sql**: Script to list all databases in MySQL server

### python-inheritance
- **7-base_geometry.py**: BaseGeometry class with integer validator method
- **tests/7-base_geometry.txt**: Test cases for BaseGeometry class

### python-serialization
- **task_00_basic_serialization.py**: Basic JSON serialization module
- **task_01_pickle.py**: Pickling custom classes module
- **task_02_csv.py**: CSV to JSON conversion module
- **task_03_xml.py**: XML serialization/deserialization module
